Welcome to parKVFinder-web server at CNPEM homepage!

 

This page is built to help you with our cavity detection and characterization web service.

parKVFinder-web server

 

Most parKVFinder functionality is available through our online web server (parKVFinder-web server).

 

The parKVFinder-web server provides a simple way to run parKVFinder with a PyMOL v2 plugin, named PyMOL parKVFinder-web Tools. To install this plugin, please refer to our group's GitLab repository.

 

Jobs availability

 

Jobs sent to parKVFinder-web server are available on the server up to 1 day after completion.

 

parKVFinder local version

 

Users may use parKVFinder through this parKVFinder-web server; however, it is also possible to install parKVFinder locally. For more information on the local version, please refer to these repository: parKVFinder or parKVFinder-win.
